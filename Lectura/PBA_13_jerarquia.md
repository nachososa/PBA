# Jerarquía

***Definición:*** Es la organización de las clases en niveles de herencia, donde las subclases derivan de clases más generales.

- Permite modelar relaciones
  - tipo "es un/a"
  - parte de

Ejemplo:

```markdown
          Animal
         /      \
     Mamifero    Ave
      /   \
Perro     Gato
```
